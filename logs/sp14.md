## What I learned today
During today's session, we began reviewing the difference between Copying and Aliasing. In Aliasing, we are referring to the same object while for copying we refer to a new object. 


For example: 
Aliasing: Memory refers to the same object, so a change in a affects a and b. 
Copying: Memory refers to different objects, so a change in a affects a but not b. 


split() treats consecutive whitespace as one separator. 
split(‘ ‘) splits on each single space (empty strings!)


We also learned how to create dictionaries and how to modify them: 

1. Start empty, add entries
prices = {}
prices['AAPL'] = 178.50
prices['GOOG'] = 141.80
prices['MSFT'] = 415.20


2. Update an existing key
prices['AAPL'] = 182.30


3. Delete a key
del prices['GOOG']

print(prices)
