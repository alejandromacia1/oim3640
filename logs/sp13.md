At the beginning of class, we reviewed some exercises about strings. While strings are immutable, we learned that lists are mutable. That is the key difference between lists and strings. sorted() returns a new sorted list (while the original remains unchanged)

stocks = [‘GOOG’, ‘AAPL’, ‘META’]
sorted(stocks) #[‘AAPL’, ‘META’,’GOOG’]
stocks #[‘GOOG’, ‘AAPL’, ‘META’] (remains unchanged)

Aliasing: Two variables can refer to the same list
a = [1,2,3]
b = a (b is not a copy of a, they are the same list)

b[0] = 99
print(a) #[99,2,3] (when b changes, a changes since they are the same list)

To make a copy: 
b = a[:]
b = list(a)
