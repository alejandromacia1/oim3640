## What I learned today
During todays class, we learned about the while loop, which keeps going until a condition becomes False. The key difference between the for and while is that for loop iterates over a known sequence and the while loop keeps going until a condition changes. The rule of thumb for these loops is that: if you know the count, use for, but if you're waiting for a condition, use while. Furthermore, we learned about the break statement, which helps us exit the loop immediately. The continue statement skips to the next iteration. The in operator checks if a character is in a string: 'e' in 'Gadsby' (Answer: False). Finally, we learned about the search patterns and how to create a function that checks if a word contains any of the letters that are given from a set of letters. For linear searches, return True early if found, and return False at the end (not indented).
