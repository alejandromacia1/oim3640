Another thing I learned was how to check if a string contains any lowercase letter using x.islower() in the if x.islower(). For the code to run properly without bugs, it has to have the same format in the for line: 

for x in s: 
    if x.islower()

Because if the code reads as: 

for x in s:
    if ‘x’.islower()


The results will change. For the first scenario the function is trying to check if a word contains a lowercase letter in any of the characters. However, the second scenario will always result in True, as the ‘x’ is already lowercase. Therefore: the variable in the for loop and the condition in the if statement must match in order for the code to actually check all the characters from the string.

If we want to check whether a letter contains a specific letter ‘x’ in lowercase, we have to write it as: 

def has_c(word):
for c in word:
if c == 'c':
return True

I was also able to learn that for other cases: 

def check_vowel(s):
    for c in s: 
        result = c in ‘aeiou’
    return result 


The result = c in ‘aeiou’ gets overwritten everytime, so the final result that is displayed will be if the last letter of a word matches or not with the condition (if the last letter is a vowel or not). A simple rule to understand this would be: 

If we write: 

for something in something: 
    variable = ... 
return variable

and return the variable after the loop, the variable will contain the value from the last iteration of the loop. 


OR Combinations
False or False → False
False or True → True 
True or False → True 
True or True → True 

AND Combinations
True and True → True
True and False → False
False and True → False
