
# Learning Log - Mini Project 1

## What did I learn during the process of creating this project? 
While creating and developing this project, I was able to improve and deepen my understanding of how to structure an interactive program that guides a user through a sequential decision-making process. Such a process began during the input collection at the beginning until reaching the final output generation section. This project allowed me to gain hands-on experience while dealing and working with loops, functions, and condition statements. Furthermore, I worked on improving formatting output for fluency and clarity, especially while dealing with multiple conditional statements throughout the whole program code. Considering that I was developing a project that is based on a topic that interests me, I was able to apply the concepts and the knowledge learned throughout these past weeks in class, reuse some functions, and connect them to a real-world scenario. 

## How did AI tools help me during the development of the project. 
While developing this project, I encountered a few formatting-related issues which slowed this process of execution, considering that when running the lines of code, these issues resulted in errors. Nonetheless, I came to realize that many of these issues were small, yet impactful mistakes. One of the main issues I encountered was incorrectly using /n instead of \n when trying to insert a new line in the printer terminal output section. Although these errors were not significant, they affected the format and the clarity of the program. 

As I sometimes doubted how to resolve these issues, considering that I was dealing with an extensive amount of lines of code, I decided to use AI tools such as ChatGPT to receive assistance and clarify these specific formatting problems. I want to highlight that I did not rely on AI tools while building the project, but instead used them as supplementary learning resources to debug formatting errors. Furthermore, I used the AI agent chat tool within the VSCode platform to optimize sections of the code once it was finished– particularly during class in 2.26. Overall, I believe that this project allowed me to understand how important attention to detail within programming is, and how AI tools can be used responsibly to enhance troubleshooting inconsistencies.
